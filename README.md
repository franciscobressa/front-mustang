Este projeto foi construido utilizando a biblioteca React do Javascript e tem o objetivo de desenvolver de forma responsiva as páginas propostas no projeto.
Ele possui duas telas responsivas, a 'Home', que é a página inicial e a página 'Tabela' que pode ser acessado através da Navbar

Para os components foi utilizado a biblioteca Reactstrap
e para os ícones a biblioteca Iconify

-----------------------------------------------------------
### `npm install`
Para baixar as dependências do projeto

### `npm start`
Para rodar o projeto em devlopment mode

-----------------------------------------------------------

Foi utilizado Styled Components para criar o Banner da aplicação

-----------------------------------------------------------
